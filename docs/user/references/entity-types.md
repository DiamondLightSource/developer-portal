# Entity Types

The catalogue is configured to ingress the nine entity types described in [Backstage Architectural Decision Record 005 (ADR005)](https://backstage.io/docs/architecture-decisions/adrs-adr005), these are:

- `Component`
- `Template`
- `API`
- `Group`
- `User`
- `Resource`
- `System`
- `Domain`
- `Location`
